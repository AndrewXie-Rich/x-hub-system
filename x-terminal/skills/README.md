# Repo-Local Skills

`x-terminal/skills/` is the active repository-local skills directory for X-Terminal.

## What It Is For

- Repo-local skills used during active terminal development
- Skills that should live with the repository instead of a user-local installation

## Boundary

- Prefer this directory for active repo-local skill development.
- User-installed skills may also exist under `~/Library/Application Support/X-Terminal/skills`.
- Do not resolve active skills from archived legacy paths.
