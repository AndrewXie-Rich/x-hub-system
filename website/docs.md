## What This Site Intentionally Avoids

- It does not mirror the whole repository.
- It does not treat every internal work order as public product copy.
- It does not lock the public story to UI details that are still changing.
- It does not expose every sensitive or still-forming implementation path.

The goal is a cleaner public entrypoint, not a second copy of the repository tree.
