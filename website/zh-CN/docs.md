## 这个网站刻意避免什么

- 不镜像整个仓库
- 不把每个内部 work order 都包装成公开产品声明
- 不把仍在变化的 UI 细节绑定成对外承诺
- 不提前暴露所有敏感或尚未成形的实现路径

目标是提供一个更干净的公开入口，而不是再复制一份仓库树。
